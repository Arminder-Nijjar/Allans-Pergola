# Allan's Landscaping & Disposal — Pergola Builder

A premium 3D pergola configurator built with React, Three.js, and Tailwind CSS.

## Features

- **Interactive 3D Design**: Visualize your pergola in real-time with Three.js
- **Multiple Layouts**: Horizontal and L-Shape configurations
- **Customizable Options**: Frame colors, louver rotation, LED lighting, screens, and walls
- **Quote System**: Submit designs for professional quotes

## Tech Stack

- **Frontend**: React 19 + Three.js + Tailwind CSS + Radix UI
- **Backend**: FastAPI + MongoDB

## Getting Started

### Frontend
```bash
cd frontend
npm install --legacy-peer-deps
npm start
```

### Backend
```bash
cd backend
pip install -r requirements.txt
python server.py
```
